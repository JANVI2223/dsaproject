# 📞 Connectify - Contact Book & Phone Directory

Connectify is a C++ console-based Contact Book and Phone Directory application developed using Data Structures and Algorithms (DSA). It allows users to efficiently manage contacts with features like adding, updating, searching, sorting, and file handling.

## 🚀 Features

- ➕ Add New Contact
- ❌ Delete Contact
- ✏️ Update Contact Details
- 🔍 Search Contact by Name
- 📱 Search Contact by Phone Number
- 📋 View All Contacts
- 🔤 Sort Contacts Alphabetically
- ⭐ Add/Remove Favorite Contacts
- 🚫 Block/Unblock Contacts
- 🔒 Lock Protected Contacts 
- 💾 Save Contacts to File
- 📂 Load Contacts from File

## 📚 DSA Concepts Used

- Arrays / Vectors
- Searching
- Bubble Sort
- File Handling
- Object-Oriented Programming
